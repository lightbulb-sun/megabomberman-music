# Mega Bomberman (Genesis) Music Swap Hack

## Changes
Swaps music track of first area with the credits
in the US version of the Genesis game
*Mega Bomberman*
in accordance with this
[romhacking.net post](https://www.romhacking.net/forum/index.php?msg=462183)
by
[fuggy_mcgee](https://www.romhacking.net/forum/index.php?action=profile;u=196596).

## Download
The latest release can be found on the
[releases page](https://github.com/lightbulb-sun/megabomberman-music/releases).
Patch the original ROM with one of the `.bsdiff`, `.ips` or `.bps` files
to create the hack.
